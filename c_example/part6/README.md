# part6 학습 안내

이 문서는 `part6`의 파일 처리와 동적 메모리 예제를 어떤 순서로 볼지 안내한다.

## 추천 순서

1. `fileDescriptor.c`, `filePointer.c`: 파일 개념 이해
2. `dynamicAllocation.c`, `dynamicAllocation2.c`, `dynamicAllocation3.c`, `dynamicAllocation4.c`: 메모리 할당 복습
3. `variableLengthArray.c`: 크기 변화와 메모리 감각 익히기
4. `scoreProcess`: 파일 입력 기반 성적 처리
5. `iotLogReport`: 프로젝트 연결형 데이터 처리 예제

## 학습 포인트

- 파일을 열고 닫는 흐름을 설명할 수 있어야 한다.
- 입력 데이터 수에 따라 메모리를 다르게 잡는 이유를 말할 수 있어야 한다.
- 단순 문법 예제를 실제 데이터 처리 문제로 연결할 수 있어야 한다.
