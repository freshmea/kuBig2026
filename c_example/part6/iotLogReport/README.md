# iotLogReport

파일 입력, 구조체 배열, 동적 메모리 할당, 포인터 정렬을 한 번에 연습하기 위한 예제다.

## 입력 형식

`sensor_log.dat`는 다음 형식을 사용한다.

```text
장치이름 온도 습도 신호세기
```

예시:

```text
sensorA 24.5 45.0 78
sensorB 31.2 68.0 52
```

## 학습 포인트

- 파일 전체 레코드 수 세기
- 구조체 배열 동적 할당
- 조건에 따라 경고 횟수 계산
- 포인터 배열을 사용해 위험 순서로 정렬
- 분석 결과를 리포트 파일로 저장

## 확장 과제

- 최고/최저 온도 장치를 추가로 출력
- 임계값을 상수 대신 사용자 입력으로 변경
- 결과를 CSV 파일로 저장
- MQTT 메시지 포맷으로 변환
