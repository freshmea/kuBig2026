# C 예제 인덱스

이 문서는 `c_example/` 폴더의 추천 학습 순서를 빠르게 안내한다.

## 추천 순서

1. `part1`: 출력, 자료형, 기본 연산자
2. `part2`: 조건문, 반복문, 배열
3. `part3`: 함수, 포인터, 정렬
4. `part4`: 분할 컴파일, CMake, 미니 프로젝트
5. `part5`: 구조체, 문자열, 표준 입출력
6. `part6`: 동적 메모리, 파일 처리, 보고서 생성

## 바로 볼 예제

- `part1/hello.c`
- `part2/scoreGrade.c`
- `part3/swap.c`
- `part4/baseballGame`
- `part5/structArray.c`
- `part6/scoreProcess`
- `part6/iotLogReport`

## 권장 사용법

- 새 개념을 배운 직후에는 예제를 그대로 실행한다.
- 그 다음에는 변수명, 입력값, 출력 형식을 한 번 이상 바꿔 본다.
- 마지막에는 같은 개념을 사용하는 다른 예제로 넘어간다.
