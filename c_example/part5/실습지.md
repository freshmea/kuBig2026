# part5 구조체/문자열 실습지

이 실습지는 `part5` 예제를 이용해 구조체, 문자열, 표준 입출력, 파일 입출력 기초를 연결해서 복습하도록 설계되어 있다.

## 목표

- 구조체와 구조체 배열을 사용하는 이유를 설명할 수 있다.
- 문자열 함수가 어떤 상황에서 필요한지 말할 수 있다.
- 표준 입출력과 파일 입출력의 공통점과 차이를 비교할 수 있다.

## 준비

- `part5/README.md`의 추천 순서를 먼저 읽는다.
- `struct.c`, `structArray.c`, `stringExample.c`, `charGetPut4.c`를 열어 본다.

## 실습 순서

1. `struct.c`에서 데이터가 어떻게 하나의 구조로 묶이는지 정리한다.
2. `structArray.c`에서 여러 개의 구조체를 다루는 방식을 확인한다.
3. `stringExample.c`에서 문자열 함수가 어떤 작업을 대신하는지 확인한다.
4. `charGetPut4.c`에서 표준 입력과 파일 입력의 차이를 설명한다.
5. part5 예제가 part6의 `scoreProcess`로 어떻게 이어지는지 3문장으로 정리한다.

## 체크 질문

- 배열과 구조체 배열은 어떤 점이 다른가
- 문자열 함수 사용 시 가장 조심해야 할 점은 무엇인가
- 표준 입출력과 파일 입출력은 코드 구조에서 어떻게 닮았는가

## 제출물

- 구조체 예제 설명 3문장
- 문자열 또는 입출력 예제에서 중요한 함수 1개와 이유
- part6으로 넘어가기 전에 다시 보고 싶은 예제 1개

## 확장 과제

- `structArray.c`를 파일 입력과 연결하는 방향을 정리한다.
- `stringExample.c`의 함수 중 하나를 직접 구현해 본다.
- `charGetPut4.c`를 결과 파일 저장 형태로 확장해 본다.
