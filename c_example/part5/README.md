# part5 학습 안내

이 문서는 `part5`의 구조체, 문자열, 표준 입출력 예제를 어떤 순서로 볼지 안내한다.

## 추천 순서

1. `struct.c`, `struct2.c`: 구조체 기본
2. `structArray.c`, `selfReference.c`: 구조체 배열과 자기참조 구조
3. `stringExample.c`, `string3.c`: 문자열 함수와 문자열 처리
4. `charGetPut.c` ~ `charGetPut4.c`: 표준 입출력과 파일 입출력 기초
5. `date`, `twoDouble`: 분리된 파일 구조의 구조체 예제
6. `unionExample.c`, `enumExample.c`: 보조 개념 정리

## 학습 포인트

- 구조체로 관련 데이터를 묶는 이유를 설명할 수 있어야 한다.
- 문자열 함수 사용 시 버퍼와 복사 범위를 의식할 수 있어야 한다.
- 표준 입출력과 파일 입출력의 차이를 예제로 설명할 수 있어야 한다.
