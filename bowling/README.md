# bowling

볼링 점수판과 프레임별 점수 계산을 연습하기 위한 미니 프로젝트다. 배열, 구조체, 함수 분리, 상태 전이를 함께 다루는 예제로 사용할 수 있다.

## 학습 목표

- 프레임 단위 상태를 구조체로 관리하는 방법을 이해한다.
- 스트라이크와 스페어가 점수 계산에 어떤 영향을 주는지 설명할 수 있다.
- 여러 함수로 게임 흐름을 나누는 이유를 설명할 수 있다.

## 실행 방법

프로젝트 루트에서 CMake로 빌드한 뒤 실행한다.

```bash
cmake -S . -B build
cmake --build build --target bowling
./build/bowling/bowling
```
