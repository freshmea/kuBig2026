# 표준 입출력과 파일 입출력 함수 정리

---

| 구분 | 단위 | 표준 입출력 (stdin/stdout) | 파일 입출력 (File / Any) | 특징 |
| --- | --- | --- | --- | --- |
| 입력 (Input) | 문자 | getchar() | getc(fp), fgetc(fp) | 한 문자씩 처리 |
|  | 문자열 | gets() (비권장) | fgets(s, n, fp) | 한 줄 단위 (fgets 권장) |
|  | 형식화 | scanf() | fscanf(fp) | %d, %s 등 타입 지정 |
| 출력 (Output) | 문자 | putchar() | putc(c, fp), fputc() | 한 문자씩 출력 |
|  | 문자열 | puts() | fputs(s, fp) | puts는 자동 줄바꿈 포함 |
|  | 형식화 | printf() | fprintf(fp) | 서식에 맞춰 출력 |
