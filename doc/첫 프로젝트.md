# 첫 프로젝트 제한 사항

---

- 기본적으로 자유 프로젝트
- 기기 - wifi 되는 MCU 에 대한 프로젝트
  - pico2w 세트 구매 개인당
- C 언어로 작성
- CMake 프로젝트로 진행

- 발표, 보고서 작성

## 필수 제출물

---

- 팀 GitHub 저장소 링크
- 실행 방법이 포함된 `README.md`
- 발표 자료
- 시연 영상 또는 현장 데모

## 권장 구성

---

- `src`, `include`, `docs` 등 역할이 드러나는 구조를 사용한다.
- 기능 설명보다 "어떤 문제를 해결하는 프로젝트인지"를 먼저 적는다.
- MQTT, UART, 센서 입력 중 적어도 하나는 실제로 연결해 본다.

## 평가 연결

---

프로젝트 채점 기준은 [프로젝트 평가 루브릭](프로젝트_평가_루브릭.md)을 따른다.
