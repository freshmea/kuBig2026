# 자료구조 예제 인덱스

이 문서는 `ds/` 폴더의 스택, 큐, 리스트 예제를 어떤 순서로 볼지 빠르게 안내한다.

## 추천 순서

1. `stack/stack1.c`: 가장 기본적인 스택 개념
2. `stack/myStack1`: 구조체로 스택 감싸기
3. `stack/myStack2`: 전역 변수 의존 줄이기
4. `stack/myStack3`: 동적 메모리 기반 스택
5. `stack/myStack4`: generic 스택
6. `queue/queue1.c`: 기본 큐 개념
7. `queue/myQueue1`, `myQueue2`: 구조체와 함수 분리
8. `queue/myQueue3`: generic 큐
9. `list/list1.c`: 연결 리스트 기초

## 학습 포인트

- 자료구조를 왜 함수와 구조체로 감싸는지 이해한다.
- 고정 크기 구조와 동적 메모리 구조의 차이를 설명한다.
- 스택과 큐의 입출력 순서를 실제 예제로 확인한다.
- generic 구조가 왜 필요한지 예제로 설명한다.
