# 04 메모장 파일 읽기/쓰기 단계별 실습

## 1. 수업 목표

- Python으로 텍스트 파일을 읽고 쓰는 데스크톱 앱을 만든다.
- `Path`, `read_text()`, `write_text()`, `encoding="utf-8"`의 역할을 이해한다.
- JS는 textarea 값을 전달하고 표시하는 최소 역할만 맡긴다.

## 2. 최종 결과물

- 데모 폴더: `memo_file_io`
- 실행: `uv run memo_file_io/main.py`
- 저장 위치: `memo_file_io/data/note.txt`

최종 동작:

1. 앱이 시작되면 Python이 `data` 폴더를 준비한다.
2. 기존 메모 파일이 있으면 내용을 읽어 textarea에 표시한다.
3. 사용자가 내용을 수정하고 저장 버튼을 누른다.
4. Python이 UTF-8 텍스트 파일로 저장한다.

---

## 3. 단계별 실습

### 단계 A. 화면부터 만들기

#### A-1. 구현

HTML에는 세 요소만 둔다.

- `textarea`
- 불러오기 버튼
- 저장 버튼

#### A-2. 문제점

화면만 있으면 데이터가 유지되지 않는다.
앱을 닫았다 열면 textarea 내용은 사라진다.

#### A-3. 해결 방향

메모 내용을 파일에 저장해야 한다.
저장 책임은 Python으로 보낸다.

강의 포인트:
- UI 상태와 영구 저장 상태를 구분한다.

---

### 단계 B. 저장 파일 경로 정하기

#### B-1. 구현

```python
BASE_DIR = Path(__file__).resolve().parent
DATA_DIR = BASE_DIR / "data"
NOTE_PATH = DATA_DIR / "note.txt"
```

#### B-2. 설명

- `__file__`은 현재 Python 파일의 위치다.
- `.resolve().parent`로 데모 폴더 기준 경로를 얻는다.
- `BASE_DIR / "data"`처럼 `/` 연산자로 경로를 조립한다.

#### B-3. 문제 해결 개념

문자열로 `"memo_file_io/data/note.txt"`를 직접 쓰면 실행 위치가 바뀔 때 깨질 수 있다.
`Path(__file__)` 기준으로 잡으면 어디서 실행해도 상대적으로 안정적이다.

---

### 단계 C. 파일 불러오기 구현

#### C-1. 구현

```python
def load_note(self) -> dict[str, str]:
    return {"text": NOTE_PATH.read_text(encoding="utf-8")}
```

#### C-2. 문제점

처음 실행하면 `note.txt`가 없을 수 있다.
파일이 없으면 `FileNotFoundError`가 발생한다.

#### C-3. 해결

`__init__`에서 폴더와 기본 파일을 준비한다.

```python
DATA_DIR.mkdir(exist_ok=True)
if not NOTE_PATH.exists():
    NOTE_PATH.write_text("처음 메모를 입력해 보세요.", encoding="utf-8")
```

강의 포인트:
- 앱 시작 시 필요한 실행 환경을 Python이 준비하게 한다.

---

### 단계 D. 파일 저장 구현

#### D-1. 구현

```python
def save_note(self, text: str) -> dict[str, str]:
    NOTE_PATH.write_text(text, encoding="utf-8")
    return {"status": "saved", "path": str(NOTE_PATH)}
```

#### D-2. 설명

- JS가 textarea 값을 Python에 넘긴다.
- Python은 파일 저장 후 결과 상태를 반환한다.
- 경로는 `str()`로 바꿔 반환한다.

#### D-3. 트릭

한글이 들어가는 수업 자료에서는 항상 `encoding="utf-8"`을 명시한다.
운영체제 기본 인코딩에 의존하지 않는 습관을 들인다.

---

### 단계 E. JS 연결

#### E-1. 구현

```js
async function saveNote() {
    const result = await window.pywebview.api.save_note(noteEl.value);
    statusEl.textContent = `저장 완료: ${result.path}`;
}
```

#### E-2. 역할 분리

- JS: textarea 값 읽기, 버튼 이벤트, 상태 문구 표시
- Python: 경로 결정, 폴더 생성, 파일 읽기/쓰기, 인코딩 처리
